# Let's build C04EL D4AW from scratch :D

Starting with the Bezier curves, which are very simple to implement.

## Usage:
- LEFT mouse button click 4 times to set all points needed for calculating curve (first 2 points are the ends of the line, next 2 are "control" points, which defines the shape of the curve).
- adjust position of each point with left mouse button when the curve is displayed.
- SPACE key to remove curve and start over.

`gem install raylib --pre`

Run:

There is executable script which you can just run:
`./run_dsl`
